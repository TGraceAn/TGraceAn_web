# TGraceAn_web

Website for Thien An Nguyen.

## Structure
- `index.html`: Main HTML, includes Tailwind CDN and Lucide icons CDN.
- `styles.css`: Custom styles extracted from inline `<style>`.
- `script.js`: App script (initializes Lucide icons) extracted from inline `<script>`.

## Run locally
No build step required. Open `index.html` in a browser.

Optional: use a simple local server to avoid any browser restrictions.
